## 시작하기
```bash
$ git clone https://github.com/choiyuri/chequer.git
$ cd chequer
$ ./gradlew build
$ ./gradlew bootRun
```

## Swagger
- URL : http://localhost:8080/swagger-ui.html
1. [사용자 생성] 사용자 API - /api/v1/signup
2. [사용자 인증] 인증 API - /api/v1/auth
3. [게시글] 게시글  API > Try it out > Access Token = 인증 Response
